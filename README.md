# ADA Cloud

ADA Cloud es una plataforma educativa inteligente basada en GitHub, Supabase e IA.

## Objetivo

Construir una plataforma escolar moderna con:

- roles institucionales
- gestión educativa
- documentos y materiales
- IA tipo NotebookLM
- horarios inteligentes
- reconocimiento de manuscritos
- reportes
- chatbot por rol
